# 🐾 Pawsome Stay

## 🔗 [Live Demo](https://milenness.github.io/pawsome-stay/)

**Pawsome Stay** is a professional web application designed for an animal shelter service. It simplifies the process of connecting pets with potential owners through a clean, minimalist interface and efficient data management.

## 🚀 Key Features

* **Responsive Multi-device Layout** — Fully adaptive design with breakpoints at 375px, 768px, and 1440px to ensure a seamless experience on all devices.
* **Dynamic Pet Catalog** — Real-time filtering by categories and asynchronous data fetching from the backend API (/api/animals).
* **Interactive UI Components** — High-performance sliders for "Success Stories" and the "About Us" section implemented with **Swiper.js**.
* **Advanced Modal Systems** — Custom-built modals for pet details and adoption forms with integrated background scroll lock.
* **Retina Ready** — All static content and background imagery are optimized for high-density displays (x2, x3).

## 🛠 Tech Stack

* **HTML5** & **CSS3** (SASS/SCSS)
* **JavaScript** (ES6+)
* **BEM Methodology** — For organized, modular, and scalable styling.
* **Vite** — Modern build tool used for optimized development performance.
* **Featured Libraries** — Includes **Swiper.js** for sliders, **SweetAlert2** for notifications, and **basicLightbox**.

## 📐 Design Specifications

* **Typography** — Precisely implemented font scales and line heights according to the Figma UI Kit.
* **Grid System** — 12-column desktop grid (1440px) and a fluid "rubber" layout for mobile devices starting from 320px.
* **Interactive States** — Custom cursor behavior and stylized hover/focus effects for all clickable elements as per the UI template.
